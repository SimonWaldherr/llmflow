How llmflow works — the prompt pipeline per record

① System Prompt
Sets the LLM's role & global instructions. Sent once — not per record.
→
② Pre-Prompt
Optional text prepended before each record's data. Good for task framing.
→
③ Input Template
Go template rendered per record. Use {{ toPrettyJSON .record }} or {{ .fieldName }}.
→
④ Post-Prompt
Optional extra instruction. Leave empty when Response Format is set — instructions are injected automatically.
→
⑤ Format Injection
When Response Format is set, llmflow appends the schema instruction + parses the JSON reply automatically.
→
⑥ Output
LLM answer stored in Response Field. With JSON format, each schema key becomes its own output column.

      1
      How to use llmflow
      A short operational guide for common tasks and UI controls.
    

    

      

        
Step 1

        
Upload or choose input

        
Start with a file in Quick Start or Files. Use the preview to inspect records and exclude columns you do not need.

      

      

        
Step 2

        
Describe the task

        
Write the task in plain English. The AI setup field should describe one job, not several unrelated tasks.

      

      

        
Step 3

        
Choose output shape

        
Use Include Input in Output, Key Column, Response Fields, and Output Fields to control what gets written.

      

      

        
Step 4

        
Run safely

        
Validate first, then try a Dry Run, and only then start a real job. Check the Jobs tab for progress and errors.

      4
      Common mistakes to avoid
      These are the most common reasons a job behaves unexpectedly.
    

    

      

        
1

        
The model returns free text

        
Make the post prompt explicit. Say that only JSON is allowed if you need structured output.

      

      

        
2

        
Too many columns are included

        
Use the preview to deselect columns that the model does not need. Fewer columns usually means better prompts.

      

      

        
3

        
Output is too large

        
Switch Include Input in Output to key or none.

      

      

        
4

        
A watcher does nothing

        
Check the directory, the pattern, and whether the watcher is active. Files must land in the watched directory, not in a subfolder.
