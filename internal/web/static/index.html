      1
      Recommended Workflow
      Shortest safe path from input to reliable output.
    

    

      

        
1. Upload + inspect

        
Upload a file, then use preview to exclude irrelevant columns.

      

      

        
2. Define structure

        
Use response format + schema to get stable, machine-friendly columns.

      

      

        
3. Validate + run

        
Validate config, dry run if needed, then start the real job.

      

    

  


  

    

      2
      Field Notes
      Controls that matter most.
    

    

      

        
Include Input In Output

        
Use key or none to keep output lean.

      

      

        
Response Fields / Output Fields

        
Response fields guide parsing. Output fields define final column order.

      

      

        
Workers

        
Higher values increase throughput but also provider load.

      

    

  


  

    

      3
      Troubleshooting
      Common causes of unexpected output.
    

    

      

        
Free text instead of JSON

        
Set response format to JSON and provide a schema.

      

      

        
Missing fields

        
Check column exclusions and output field overrides.

      

      

        
Watcher not firing

        
Verify watch directory, pattern, and watcher active status.
